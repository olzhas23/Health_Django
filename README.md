## Small Django App to help Docotors and Patients to communicate and share notes

Doctor portal. Can:
	1. Select Patients
	2. See Patients (name, ssn, notes from last apointement)
	3. Post new notes 
Patients. Can:
	1. See current doctor
	2. See insurance info
	3. Change insurance info
	4. See notes
	5. Reply to Dr. notes
